🎮 Card Matching Game

📋 How to Play:

• Find matching pairs by flipping 2 cards per turn

• First player to find 4 pairs wins!

• ⚠️ Joker card costs you -1 point!

• If you find a match, you get another turn!

👥 Two Player Mode

Player 1: 0 pairs
Player 1's Turn
Player 2: 0 pairs
Player 1: Choose 2 cards to flip

🤖 vs Computer Mode

You: 0 pairs
Your Turn
Computer: 0 pairs
